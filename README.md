# Fast Formulas Compensation Default and Override Oracle Fusion HCM Mérito

Repositorio público con **Fast Formulas de Oracle HCM** enfocadas en procesos de compensación y mérito.

##  Contenido
- Fórmulas de apertura y cierre de procesos de compensación.
- Cálculo de bonos por mérito.
- Ejemplos de override y default en Oracle HCM.
- Documentación de uso y buenas prácticas.

##  Uso
1. Importa las fórmulas en tu entorno de Oracle HCM.
2. Ajusta parámetros según las necesidades de tu organización.
3. Verifica resultados en pruebas antes de mover a producción.
